
# 🚀 Pic2Page

AI-Powered Image & Document Analysis Application built using Google Gemini Vision, Streamlit, and Python.

## 📖 Overview

Pic2Page is a multimodal AI application that allows users to upload images or PDF documents and ask questions in natural language.

Powered by Google Gemini Vision, the application analyzes visual content, extracts information, summarizes documents, identifies vehicle damage, and provides intelligent responses based on user prompts.

## ✨ Features

- AI-powered image analysis
- PDF and document analysis
- Vehicle damage assessment
- Research paper summarization
- Handwritten text extraction
- Prompt-based image understanding
- Multi-language explanations
- Interactive Streamlit interface

- ## 🎯 Demo Use Cases

### 🚗 Vehicle Damage Analysis
- Detect damaged vehicle parts
- Identify affected areas
- Generate repair insights

### 📄 Research Paper Analysis
- Extract key points
- Summarize content
- Explain complex concepts in simple language
- Translate explanations into native languages

### 📝 Handwritten Document Analysis
- Read handwritten text
- Extract important information
- Answer user questions
